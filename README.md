# Contact List

Contact List is simple CRUD app written with Laravel 8 & Vue.js.

## Installation

After cloning:

Create Data base and .env file

- install composer
```bash
composer install
```

- instal NPM and run Dev
```bash
npm install && npm run dev
```

Start Laravel
```bash
php artisan serve
```
